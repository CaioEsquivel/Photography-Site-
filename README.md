# Photography-Site-
Projeto focado em CSS, responsividade e organização visual, com atenção à usabilidade e experiência do usuário.

Este projeto foi desenvolvido com foco em design moderno, responsividade e experiência do usuário. A interface foi estruturada para se adaptar a diferentes tamanhos de tela, garantindo navegação fluida tanto em dispositivos móveis quanto em desktop.

Foram aplicadas técnicas avançadas de CSS para controle de layout, alinhamento e efeitos visuais, como o uso de zoom em imagens sem comprometer a estrutura da página, proporcionando mais interatividade e dinamismo ao conteúdo.

O desenvolvimento priorizou:
Organização e reutilização de classes
Layout limpo e funcional
Otimização para mobile
Atenção aos detalhes visuais
Performance e usabilidade

O resultado é uma interface equilibrada, intuitiva e visualmente atrativa, pensada para valorizar o conteúdo e oferecer uma experiência agradável ao usuário.
